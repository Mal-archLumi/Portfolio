## 🔧 QUICK TROUBLESHOOTING

### Site Not Indexed?
1. Check robots.txt allows crawling
2. Verify no noindex tags
3. Request indexing in Search Console
4. Wait 1-2 weeks

### OG Image Not Showing?
1. Clear Facebook cache (Sharing Debugger)
2. Verify image is 1200×630
3. Check image URL is absolute
4. Ensure image < 1MB

### Low Performance Score?
1. Compress images
2. Enable lazy loading
3. Check Netlify compression
4. Minimize third-party scripts

### Structured Data Not Recognized?
1. Validate JSON-LD syntax
2. Check for JavaScript errors
3. Test with Rich Results Test
4. Wait 24-48 hours for re-crawl
